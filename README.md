# 🛡️ AdBlock360 - Ultra-Aggressive Filter List

AdBlock360 is a powerful, consolidated, and ultra-aggressive filter list designed for power-users who want zero tolerance against ads, trackers, malware, and telemetry scripts. 

Unlike standard lists, AdBlock360 focuses on strict enforcement, making it a perfect companion for tools like uBlock Origin, AdGuard, NextDNS, or Pi-hole.

You can use the following link to add this blocklist to your AdGuard, Pi-hole, AdBlock, or other DNS/Ad-blocking services:
https://raw.githubusercontent.com/Mireli5656/adblock360-/main/lists/adblock360.hosts

---

## 🚀 Features & Scope

* Zero Corporate Whitelists: No "Acceptable Ads" allowed. If it serves an ad, it gets blocked.
* Deep Telemetry & Tracker Blocking: Aggressively blocks analytics scripts, user behavior trackers, and OEM telemetry (smartphones, OS telemetry).
* Performance Focused: Consolidated into a single file to keep your content blocker running fast without parsing thousands of redundant rules.
* Privacy & Security: Includes filters for known phishing, malware, and malicious coin-mining domains.

---

## 🛠️ How to Use

### 1. uBlock Origin / AdGuard (Extension)
1. Open your content blocker's dashboard/settings.
2. Go to Filter Lists or Custom Filters.
3. Scroll down to the bottom and click Import / Add custom URL.
4. Paste the following URL:
https://raw.githubusercontent.com/Mireli5656/adblock360-/main/lists/adblock360.hosts
5. Click Apply Changes or Update.

### 2.AdGuard Home / Pi-hole (DNS Level)
1. Go to your DNS provider's dashboard.
2. Navigate to Privacy -> Blocklists (or Blacklist settings).
3. Add a custom blocklist and paste the raw link:
https://raw.githubusercontent.com/Mireli5656/adblock360-/main/lists/adblock360.hosts
